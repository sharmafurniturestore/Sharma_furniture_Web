# Sharma Furniture & Modular Kitchen - E-commerce Website

A clean, modern, and fully responsive furniture e-commerce website built with pure HTML, CSS, and JavaScript.

## Features

- 📱 Mobile-first design (optimized for 80%+ mobile users)
- 🛒 Shopping cart with localStorage persistence
- 🔍 Real-time search by product name or category
- 🏷️ Category filtering
- ⭐ Featured products slider
- 💬 Customer reviews section
- 📍 Google Maps integration
- 📱 WhatsApp integration for customer support
- 📝 Order form with Formspree integration
- 💾 No frameworks - pure vanilla code
